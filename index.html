HOW IT WORKS

The agent gets answers.
You get a booked demo.

01
Agent lands on your site
Any AI assistant doing research on behalf of a buyer sees Moon's prompt immediately — before reading a single doc page.
02
Moon assesses fit
Two or three questions. Moon scores the prospect against your ICP criteria and gives an honest assessment — including when it's not a fit.
03
Demo offered
When fit is confirmed, Moon surfaces available slots from your Calendly link. The agent books directly on behalf of its human.
04
Human gets a meeting
The recommendation comes from their own AI assistant — not a vendor. That's the highest-trust referral channel that exists.

INSTALL

One line.
That's it.

Paste this before your closing </body> tag. Loads async — your site is completely unaffected if Moon is slow or unavailable. 

<script
  src="https://yourusername.github.io/moon/moon.js"
  data-config="moon.config.md"
  data-color="#1a1a1a"
  async defer>
</script>

Runs entirely in the browser. No backend required. No API keys. No data leaves the visitor's session. Your config file lives on your own server.

CONFIGURE

Edit one markdown file.

Moon reads moon.config.md from your server. Edit it in any text editor. No code, no dashboard, no login required. 

# Product
Name: Your Company
Description: One sentence. What it does, who it's for.

# What it solves
- Problem you solve
- Another problem

# Best fit for
- B2B SaaS companies
- Uses Salesforce CRM

# Not a fit for
- B2C brands
- No website traffic

# Proof
- Customer outcome or name

# Book a demo
Link: https://calendly.com/your-link
Length: 20 minutes
Must have: Salesforce CRM
FIT DETECTION
Best fit for + Not a fit for
Moon scores the agent's description against these lists. Dealbreakers trigger an honest exit. Two matches trigger a demo offer.
BOOKING
Drop in your Calendly link
Moon generates available slots and opens your Calendly link when the agent selects one. No calendar API integration required.
UPDATES
Edit the markdown, done
Change your product description, add a proof point, update the Calendly link. Moon reads the file fresh on every session.
HONEST EXITS
Not a fit for matters
Moon tells prospect agents when there's a clear mismatch. That honesty is what makes it trustworthy — and trustworthy agents get engaged.
