HOW TO SET UP ANALYTICS
01
Open Google Apps Script
Go to script.google.com, create a new project called "Moon Analytics", and paste the contents of moon-analytics.gs.
02
Deploy as web app
Click Deploy → New deployment → Web app. Set Execute as: Me. Who has access: Anyone. Copy the URL.
03
Add to moon.js script tag
Add the URL to your script tag:
data-analytics="YOUR_URL"
04
Paste URL above
Paste your Apps Script deployment URL in the banner above and click Connect. Sessions will appear here as agents interact with Moon.
